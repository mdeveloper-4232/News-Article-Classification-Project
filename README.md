# News-Article-Classification-Project


##  Project Description

This project builds a machine learning model to classify news as **Fake** or **Real** using text analysis.

##  How It Works

1. Load fake and real news dataset
2. Convert text into numbers using TF-IDF
3. Train Logistic Regression model
4. Test accuracy
5. Save trained model

##  Dataset

Fake and Real News Dataset from Kaggle:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

##  How to Run

1. Install Python libraries:
   pip install pandas scikit-learn nltk joblib

2. Run program:
   python train.py

##  Result

Model accuracy: ~98%
